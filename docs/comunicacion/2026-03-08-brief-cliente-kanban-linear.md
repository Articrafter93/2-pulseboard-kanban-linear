## Resumen operativo
- Construir `Pulseboard` como producto SaaS colaborativo tipo Linear.
- Stack oficial: Next.js App Router + TypeScript + PostgreSQL + Prisma + Socket.IO + Redis + Clerk Organizations.
- Entregables exigidos: experiencia premium responsive, Docker local, README completo, deploy limpio y demo usable.

## Requerimientos funcionales consolidados
- Workspaces multiempresa, proyectos, columnas, ciclos y tareas.
- Tareas con descripcion, prioridad, etiquetas, fecha limite, responsable, subtareas, comentarios, historial y adjuntos.
- Vistas tablero, lista y calendario.
- Realtime con presencia, edicion, notificaciones y estados optimistas.
- Dashboard ejecutivo, workload, vencidas/bloqueadas/completadas y reportes exportables.

## Requerimientos UX/UI consolidados
- Dark mode nativo.
- Minimalismo, jerarquia clara, espacio negativo y sensacion premium.
- Animaciones cortas y utiles.
- Navegacion lateral limpia, topbar ligera y atajos visibles.
